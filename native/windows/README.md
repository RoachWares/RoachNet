# RoachNet Native Windows Scaffold

Target:

- Windows 11 x64
- WinUI 3
- Separate setup app and main app

Projects:

- `RoachNet.Setup/`
  Native setup and onboarding flow.
- `RoachNet.App/`
  Native main application shell.

This scaffold is the Windows counterpart to the macOS SwiftUI package. It is intentionally focused on:

- clean window structure
- setup-first product flow
- RoachNet-native visual language
- room for RoachClaw, runtime, and knowledge surfaces
