# RoachNet Native Linux Scaffold

Target:

- Ubuntu desktop
- Bazzite / Fedora Atomic style environments
- GTK4 + libadwaita

Workspace layout:

- `roachnet-design/`
  Shared tokens and small reusable surface builders.
- `roachnet-setup/`
  Native setup and onboarding shell.
- `roachnet-app/`
  Native main application shell.

This scaffold is intended to evolve toward Flatpak-first packaging, with room for `.deb` or AppImage outputs later.
